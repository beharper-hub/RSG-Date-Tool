How-To Guide

Quick reference for using each calculator. Arithmetic support only — verify against policy, prescription directions, adjudication, and pharmacist guidance.
← Back to Main Suite

General Use

Calculate
Runs the calculator for that card.
Copy
Copies the displayed result text.
Reset
Clears the card back to default values.

The main Calculator uses AC instead of Reset because it behaves like a standard calculator.

Calculator

What it is for

Quick standard or scientific calculations inside the suite.

Fields / controls

Use on-screen buttons or keyboard inputStandard mode is defaultScientific mode adds advanced functions

How to use

  1. Use the number/operator buttons to enter a calculation.
  2. Tap = to calculate.
  3. Use AC to clear everything or Backspace to remove the last entry.
  4. Use Scientific only when you need square root, trig, logs, powers, or pi.

Refrigerated Shipment Checker

What it is for

General workflow arithmetic support.

Fields / controls

Proposed delivery dateApply UPS holiday logicAvoid weekend exposure logic

How to use

  1. Enter the proposed delivery date.
  2. Leave UPS holiday logic and weekend exposure logic checked unless intentionally reviewing without those safeguards.
  3. Tap Check to flag holiday, weekend, Monday, Friday, or cold-chain timing concerns.
  4. Reset clears only the date/result while keeping both safety checks on.

General / Liquid Day Supply

What it is for

General workflow arithmetic support.

Fields / controls

Calculation typeTotal quantityUnits used per dayTotal mL dispensedmL per doseDoses per dayStarting dose / units per dayChange byTarget dose / units per dayChange everyInterval unitDirection

How to use

  1. Choose General units, Liquid mL, or Titration schedule.
  2. For General units: enter total quantity and units used per day.
  3. For Liquid mL: enter total mL, mL per dose, and doses per day.
  4. For Titration schedule: enter starting dose, change amount, target dose, interval, direction, and optional quantity.
  5. Choose rounding where applicable, then tap Calculate.
