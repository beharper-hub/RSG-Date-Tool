General Use

Calculate
Runs the selected module.
Copy
Copies the displayed result text, including the main result and supporting details.
Reset
Clears the card back to default values.
Result boxes now show the answer first.
The top line is the main result. Supporting math, assumptions, and warnings appear underneath so the answer is easier to scan during workflow interruptions.

Calculator histories keep the 5 most recent entries. Once history is full, the newest result replaces the oldest. The main Calculator uses AC instead of Reset because it behaves like a standard calculator.

Customize Layout

What it is for

Lets you rearrange calculator cards so the tools you use most can sit closer to the top. This is saved only on the current device/browser.

How to use

  1. Tap Customize Layout.
  2. Use Up, Down, or Top in the overlay list.
  3. The card order updates immediately and saves automatically on that device.
  4. Tap Done or × to close the overlay.
  5. Use Reset Layout to clear the saved order and return to the default order.
Layout note:
This does not change calculations. It only changes card order in the browser using local storage.

Calculator

What it is for

Quick standard or scientific calculations inside the suite.

Fields / controls

Keyboard input Standard mode Scientific mode Backspace AC Recent Calculations Clear History

How to use

  1. Click the calculator, then use the on-screen buttons or keyboard.
  2. Tap = or press Enter to calculate.
  3. After a completed calculation, pressing a number starts a new equation automatically.
  4. After a completed calculation, pressing +, −, ×, or ÷ continues from the previous answer.
  5. Use Backspace to remove the last entry.
  6. Use AC to clear the current calculation.
  7. Use Scientific only when you need square root, trig, logs, powers, or pi.

Fill Date / RTS

What it is for

Estimates the eligible next fill date using the last fill date, days supply, and selected threshold.

Fields / controls

Last fill date Days supply Threshold Control / C-II? Auto-use 100%

How to use

  1. Enter the last fill date.
  2. Enter the days supply.
  3. Choose the refill threshold.

  5. Tap Calculate to display the eligible next fill date.

Standard Day Supply

What it is for

Day-supply support for tablets/capsules, liquids, titration schedules, daily-dose day-supply math, and estimated medication remaining.

Fields / controls

Calculation type Tablets/Capsules Liquid mL Simple Titration Multi-Step Titration Daily Dose Medication Remaining Qty Dispensed Used Per Day mL Dispensed mL Per Dose Doses/Day Starting Dose Change by Target Dose Adjust Every Direction Step Dose Step Duration Unit(s) Per Day Optional Quantity Coverage Estimate Rounding Last Fill Date As Of Date Expected Remaining

How to use

  1. Choose Tablets/Capsules, Liquid mL, Simple Titration, Multi-Step Titration, Daily Dose, or Medication Remaining.
  2. For Tablets/Capsules: enter Qty Dispensed and Used Per Day.
  3. For Liquid mL: enter mL Dispensed, mL Per Dose, and Doses/Day.
  4. For Simple Titration: enter Starting Dose, Change by, Target Dose, Adjust Every, interval unit, direction, and optional Qty Dispensed.
  5. For Multi-Step Titration: enter each active step with its dose/use amount and duration. Leave unused step rows blank.
  6. For Daily Dose: enter Qty Dispensed and Daily Dose to estimate day supply.
  7. For Medication Remaining: enter Qty Dispensed, Daily Dose, and Last Fill Date. Leave As Of Date blank to use today.
  8. Enter optional Qty Dispensed when you need the calculator to estimate coverage and flag shortages.
  9. Choose rounding when the selected mode uses rounding, then tap Calculate.
Multi-Step Titration is for staged schedules.
Use it when the directions move through multiple dose levels or time blocks instead of one simple increase/decrease. The calculator totals the scheduled use across the entered steps, estimates coverage when quantity is provided, and assumes any remaining quantity continues at the final entered dose.

Example: Step 1 uses 1 tablet daily for 7 days, Step 2 uses 2 tablets daily for 7 days, and Step 3 uses 3 tablets daily afterward. Verify the actual prescription directions and pharmacist guidance before applying the result.

Daily Dose:
Use when you know the quantity dispensed and expected daily dose but want the day supply. Example: 180 tablets ÷ 6 tablets per day = 30 days.
Medication Remaining:
Estimates expected quantity remaining, expected days remaining, and estimated run-out date from quantity dispensed, daily dose, and last fill date. This is an estimate only; actual supply can differ because of adherence, dose changes, lost medication, early starts, late starts, or prescriber-directed changes.

Patterned Dosing

What it is for

Day-supply support for uneven weekly tablet/capsule schedules, such as a base daily dose plus extra doses on selected days.

Fields / controls

Qty Dispensed Daily Base Dose Extra Dose Extra Days/Week Rounding

How to use

  1. Enter Qty Dispensed.
  2. Enter the Daily Base Dose.
  3. Enter the Extra Dose amount, if any.
  4. Enter how many days per week the extra dose is used.
  5. Choose rounding, then tap Calculate.

Example: 1 tablet daily plus 1 extra tablet on Monday, Wednesday, and Friday equals 10 tablets per week.

Eye/Ear Drop Day Supply

Estimates day supply for eye or ear drops using mL, drops per mL, dose, frequency, and treated eyes/ears.

Fields / controls

mL Dispensed Drops/mL Drops Per Dose Doses/Day Eyes/Ears Treated

How to use

  1. Enter mL Dispensed.
  2. Drops/mL defaults to 15.
  3. Adjust Drops/mL only when workflow or product guidance supports a different value.
  4. Enter Drops Per Dose and Doses/Day.

Warfarin Weekly Dose

What it is for

Totals a weekly warfarin schedule and optionally estimates tablets per week and day supply when tablet strength and quantity are entered.

Fields / controls

Monday mg Tuesday mg Wednesday mg Thursday mg Friday mg Saturday mg Sunday mg Tablet Strength Optional Qty Dispensed Optional

How to use

Arithmetic support only. Does not recommend dose changes or interpret INR.
